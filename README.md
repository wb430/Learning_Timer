# Learning_Timer
个人学习计时器与效率分析
选择要学习或自习的科目，支持手动添加
点击开始学习开始计时，切屏超过3s标记一次分心，分心原因为窗口失焦，并记录分心时长，同时学习时间也在记录
当返回主窗口时，更新分心次数和专注度评分
点击结束学习，会重置时长
点击历史记录可以查看学习记录包括id，学习科目，开始时间（YYYY-MM-DDThh:mm:ss），学习时长，分心次数，分心时间，专注度评分
在历史记录面板可以通过复选框根据学习科目筛选学习记录，点击单条学习记录可以查看详细信息
在学习报告面板可以根据时间生成本周/上周/本月/上月/全部的学习报告
在设置面板里面可以是否启用分心检测，设置分心检测时间阈值，删除学习记录，恢复默认设置，保存设置
由于本软件仅支持通过切屏检测分心，支持手动添加分心记录+分心原因
